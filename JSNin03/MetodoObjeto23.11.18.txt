Objeto - Conjunto de Propriedes

> var myvar = function() {
return 'variável myvar';
}
>myvar()
'variável myvar'

Um objeto pode ter as propriedades, as propriedades podem ter um nome ou valor
e esse valor pode ser um valor primitivo ou um outro objeto.
Quando atribuímos a uma prorpiedade de objeto uma função, temos então um Método.

> var pessoa = {nome: 'Thalita', idade: 23, altura:1.70, peso: 70};

ATRIBUINDO UMA NOVA PROPRIEDADE
> pessoa.cor = 'caucasiano';
> pessoa 
{nome: 'Thalita', 
idade: 23, 
altura:1.70, 
peso: 70,
cor: 'caucasiano'};
>pessoa.cor
'caucasiano'}

Os métodos do objeto, podem ser basicamente ações que atribuímos a variável.

> pessoa.andar = function(){
return 'Pessoa andando';
}


> pessoa.cor = 'caucasiano';
> pessoa 
{nome: 'Thalita', 
idade: 23, 
altura:1.70, 
peso: 70,
cor: 'caucasiano'
andar: [Function};}
- Acima temos todas as propriedade mais o método andar.

> pessoa.andar()
'Pessoa andando';

--------------------------------------
> pessoa.idade
23
> pessoa.aniversario = function(){
pessoa.idade++;
}
> pessoa 
{nome: 'Thalita', 
idade: 23, 
altura:1.70, 
peso: 70,
cor: 'caucasiano'
andar: [Function]
aniversario: [Function]}

>pessoa.andar()
undefined

Observe a idade no objeto pessoa.

> pessoa

A função aversário acrescenta um valor na idade.

> pessoa.sobrenome = 'Mariano'

> pessoa 
{nome: 'Thalita', 
idade: 23, 
altura:1.70, 
peso: 70,
cor: 'caucasiano'
andar: [Function]
aniversario: [Function]
sobrenome: 'Mariano'}

> pessoa.nomeCompleto = function(){
return pessoa.nome + ' ' + pessoa.sobrenome
}


> pessoa 
{nome: 'Thalita', 
idade: 23, 
altura:1.70, 
peso: 70,
cor: 'caucasiano'
andar: [Function]
aniversario: [Function]
nomeCompleto: [Function]}

> nomeCompleto()
'Thalita Mariano'
