Condicional Ternário:

Ele substitui o if.
Ele cria uma forma mais fácil, prática ou menor com o operador ternário.
Teremos uma condição:

condição ? true : false

condição + interrogação + o resultado se for true : o resultado se for false

> 1 === 2 ? true : false;
false

> if (1 ===2) {true;} else {false};
false
