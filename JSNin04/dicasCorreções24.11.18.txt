Quando criamos uma função LITERAL, não há necessidade de atribuirmos 
ponto e vírgula no final.

function lalala(){}

Mas quando atribuímos um valor a uma variável ( uma função é um valor
normal em JavaScript), ou a uma propriedade em um objeto, PRECISAMOS TERMINAR
COM PONTO E VÍRGULA.

Para descrobrirmos o equivalente boleano usamos a exclamação 2x.

> !!undefined
false (o valor dele é falsy)
>!!1
true

var is Truthy = function(param){
	if(param){
	return true;
};
return false;
};


------------------------------
TERNÁRIO

var is Truthy = function(param){
return param ? true : false;
};

------------------------------
Utilizando o Equivalente Boleano <3


var is Truthy = function(param){
	return !!param;
};

-----------------------------------
Podemos passar funções por parâmetros

iTruthy(function(){});

-------------------------------------

Quando tratamos de alguma propriedade de um objeto, procuramos utilizar métodos(funções) para
isso passando parâmetros para o método para que tratarmos as propriedades do objeto.
