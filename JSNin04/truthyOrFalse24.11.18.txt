truthy e falsy

Quando precisamos avaliar um calor, por exemplo com o if,
truthy é todo valor que quando convertido para boleano é representado por true
false é todo valor que quando convertido para boleano é representado por false

ex.

>1 === 'b'
false
>1
1
Para saber se esse valor é ou não boleano..
Testamos com if

> if( 1 ) { teste = true } else {teste = false;}
> teste
true

> if( 0 ) { teste = true } else {teste = false;}
> teste
false

O zero mesmo sendo um número, ele retorna boleanamente false.
Valores FALSY

-	undefined
-	null
-	NaN
-	0
-	-0
-	'' ou ""
-	false

> if( '' ) { teste = true } else {teste = false;}
> teste
false

> if( '0' ) { teste = true } else {teste = false;}
> teste
true

> if( undefined ) { teste = true } else {teste = false;}
> teste
false

> if( null ) { teste = true } else {teste = false;}
> teste
false

> if( NaN ) { teste = true } else {teste = false;}
> teste
false

> if( -0 ) { teste = true } else {teste = false;}
> teste
false

> if( false ) { teste = true } else {teste = false;}
> teste
false
Valores TRUTHY 
Todos os outros valores.

Arrays e Objetos retornam true

> if( {} ) { teste = true } else {teste = false;}
> teste
true

> if( [] ) { teste = true } else {teste = false;}
> teste
false

Existe uma outra forma de verificar o valor boleano sem utilizar o if, é utilizando duas vezes o sinal de
exclamação.

Descobrir representação boleana:
!!

Um único sinal de exclamação, converte para boleano e inverte o valor.
>!true
false

>!false
true

>!!"true"
true

>!!""
false

>!!undefined
false

>!!0
false
