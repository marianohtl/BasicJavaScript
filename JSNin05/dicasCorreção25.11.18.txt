notações de arrays e objetos, usando ponto 
(quando temos um nome válido de variável), 
o formato para retornarmos os nomes acima

Em JavaScrip um array é uma mentira, ele não existe.
Arrays nada mais são que objeto que tem os índice numerado.

> var arr = [1,2,3];

> var arr = {
	"0":1,
	"1":2,
	"2":3
}

a notação de chamada do array, arr.1 está errada ,pois não podemos chamar
uma variável com números, somente com letras, underscore ou cifrão

>> arr[1]


Com objetos podemos usar a mesma notação de array pois arrays nada mais são do que meros objetos.

>> arr["0"]

Logo sempre que não soubermo o valor da propriedade do objeto ou esse valor for uma string,
precisamos utilizar a notação de array.
