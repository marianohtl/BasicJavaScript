O Operador Módulo - %

O operador de módulo retorna para gente o resto 
de uma divisão entre dois operandos (inteiro).

3 / 3 = 1
3 % 3 = 0
5 % 2 = 1

Utilidades

- números pares e ímpares
Números de 0 a 20 que são pares.

var counter = 0;
 
while(counter <= 20){
	counter % 2 === 0 ? console.log(counter) : '';
	counter++;
}
