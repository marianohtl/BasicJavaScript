OPERADORES DE IGUALDADE / RELACIONAIS
Servem para testar se os valores são iguais ou diferentes a outros valores,
sempre retornando true ou false.

== igual a
!= diferente de

> 1 == 2
false

> 1 != 2
true

Testando Variáveis

> var x = 0
> x == 0
true

> 'thalita' == 'thalita
true
> var nome = 'thalita'
> thalita == nome
true

> var um = '1'
> var numum = 1

> um == numum
true

O caso acima ocorro pois o JavaScript faz conversões de strings, JavaScript é uma linguagem com tipagem fraca e tipagem dinâmica.
> 1 == '1'
true

O JavaScript converte o tipo para um número, logo torna a expressão verdadeira.


=== igual a, e do mesmo tipo (esse operador teste também o tipo)
!== diferente, mas do mesmo tipo

> 1 === '1'
false

> 1 !== '1'
true
