Variável é um nome simbólico para um tipo de valor.

  (nome para variável)
> var x
> x
undefined

Atribuindo valor à variável:

> x = 0
> x
0

Posso mudar os valores da variável.

> x = 2
> x
2

Tipos de valores para variáveis.

Tipo de número float ou número real. (conhecido na matemática como decimais)

> x = 1.2
> x 
1.2

Array é uma lista de valores.

[] array


>> var numeros = []

Definindo valores.

>> numeros = [1,2,3,4,5,6];

Acessando os valores dos arrays.

>> numeros[0]
1
>> numeros[3]
4

No JavaScript não existe exatamente arrays, arrays nada mais são do que objetos.

> pessoa
{altura: 1.87, peso: 80};

> pessoa['peso']
80

Logo podemos usar tanto a notação de objeto quanto a notação de array.

> pessoa.peso
80

> pessoa['peso']
80
