Github AULA 02-011

atualizando um repositório 

copiar URL do repositório

> git clone end.ssh

Acessar a pasta do curso.

comando:
cd git branch (mostra as branchs que nós temos)

> git remote add upstream end.url 

Atualizando o repositório local com o repositório principal.
( ele baixa tudo que tem o repositório upstream para cá)
> git fetch upstream

> git merge upstream/master

comando:
> git checkout -b novabranchelalala
(ele cria uma nova branch e já troca para ela)

> git checkout master 
(volta para a branch master)

> git checkout novabranchlalala
(ele vai para a branch novabranchlalala)

>git status
(mostra que o documento foi ou não modificado)

> git commit -m "comentário"
(comentário para o repositório)

> git push origin novabranchlalala
envia todas as mudanças para a nova branch
